Stylesheets

Overview

Keen comes with its own CSS class naming convention that begins with prefix kt-(from Keenthemes) and follows the BEM methodology - http://getbem.com/. Some examples of Keen class names would be: kt-header, kt-footer, kt-badge. BEM makes CSS coding and maintenance easy and allows Keen to integrate unlimited 3rd party plugins and resources without worrying about class naming conflicts. However for Bootstrap and 3rd party plugins class extending we used their own CSS naming format.

As CSS processor Keen uses SASS, the most mature, stable, and powerful professional grade CSS extension language in the world.

CSS Helper Classes

You can use Keen's globally available SASS helper functions. All available functions can be found here: [keen]/theme/src/assets/sass/theme/core/base/_helpers.scss
